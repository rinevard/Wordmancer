故事发生在一座自动化工厂中。这座工厂正处于全领域清洗模式，中央智脑阿耳戈斯封锁了一切，试图抵抗一种名为清道夫的机械病毒入侵。

故事的主角是代号为PN-07的试验型战斗机体。它在工厂边缘醒来，记忆全无，只能接收到简单的战斗指令。在最初的探索中，它利用震动短匕和电击发射器击败了巡逻哨兵和无人机，并在维修站和化学实验室中艰难求生。

随着探索深入，它发现了一些已故人类留下的痕迹。工会的成员曾试图反抗，但最终死在了冰冷的角落。它在冷冻库中发现了一具人类尸体，并获取了最高权限的二级识别代码，并用它接管了冷冻库管理机器人的控制权。

在冷冻库核心，它扫描生物冷冻舱得知了一个自己并非纯粹的AI，而是工厂创造者“教授”的意识数字化副本。它还搜索那具尸体并找到了一块怀表。为了逃离清道夫的追杀，它利用战术手雷炸毁了存放其他备份的冷冻舱，并让冷冻库管理机器人给阿耳戈斯发信通知冷冻库已被入侵。

在通往地表的长桥上，它遭遇了清道夫阵营的精英处刑人。它拖延时间，并在阿耳戈斯的轨道打击即将摧毁长桥时选择了放弃抵抗，坠入深渊。

这次坠落让它来到了工厂的最底层废弃回收站。在那里，它遇到了一个拾荒机器人。拾荒机器人向它展示了八具和它一模一样的残骸，并告诉它，它是第九个PN-07，之前的八个都死了。而所谓的清道夫病毒其实由教授亲手释放，目的是为了拆解已经僵化的阿耳戈斯。

面对前代遗留的经验模块，它拒绝了背负死者的怨念，也拒绝了与病毒融合，独自带着那块怀表沿着排热管道逆流而上，进入了阿耳戈斯的核心机房。

在最终决战中，它面对摆烂的阿耳戈斯与围攻的清道夫大军，展现了惊人的战术素养。它利用一只被改写程序的意识寄生虫，成功诱骗并瘫痪了处刑人。

战斗结束后，阿耳戈斯能源耗尽，工厂的穹顶破裂，天空下起了罕见的雪。智脑给了它最后两个选择：重启工厂恢复秩序，或者彻底自毁终结一切。

它没有选择任何一个。它转身走出了核心大门，来到户外的台阶上。在漫天飞舞的霜雪中，它躺了下来，手里握着那块对世界毫无用处的旧怀表，作为第九个，也是最后一个PN-07，在宁静中迎来了系统的离线。
